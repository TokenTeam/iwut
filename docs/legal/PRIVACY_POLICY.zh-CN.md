# 掌上吾理隐私政策

生效日期：2026 年 6 月 19 日

TokenTeam（以下简称“我们”）重视你的个人信息和隐私保护。本政策适用于我们向你提供的“掌上吾理”移动应用、网站、安装包、更新服务及相关功能（以下简称“本应用”或“服务”）。

本政策用于说明我们如何收集、使用、存储、共享和保护你的信息，以及你如何管理相关信息。请在使用本应用前仔细阅读。如果你不同意本政策，请停止使用本应用中涉及个人信息处理的功能。

本应用为校园工具类开源软件，不提供广告投放、商业营销或基于个人特征的定向推送功能。学校账号密码、校园网密码、学号、姓名、一卡通号、课程表、Wi-Fi 或网络状态等信息可能对你具有较高敏感性，请你谨慎授权、妥善保管设备，并优先从官方渠道获取安装包。

## 二、敏感个人信息提示

在使用学校账号绑定、自动登录、校园网辅助登录、课程导入、考试安排导入、日历同步等功能时，本应用可能处理账号密码、学生身份信息、课程安排、考试安排、Wi-Fi 或网络状态等信息。其中，账号密码、特定身份信息、行踪或网络状态相关信息在特定场景下可能属于敏感个人信息或重要凭据。我们仅在你主动使用相应功能、授权相应权限或为了完成你请求的操作时处理这些信息。

## 四、我们如何存储和保护信息

1. 学校统一身份认证密码、校园网密码等敏感凭据会尽量使用系统安全存储能力保存，例如 iOS Keychain、Android Keystore 或 Expo SecureStore/原生安全凭据模块。
2. 课程表、考试安排、学号、姓名、学院、培养层次、应用设置、公告状态等信息主要保存在设备本地。
3. WebView 访问学校系统时可能产生 Cookie、缓存和会话数据。本应用在解绑、导入结束、清理缓存等场景会尝试清理相关缓存或 Cookie。
4. 本地日志默认保存在设备本地，并限制文件大小和保留数量。你可以在设置中清除缓存或主动导出日志。
5. 我们会采取合理的技术和管理措施保护信息安全，但互联网传输、学校系统、设备系统、第三方 SDK 和用户使用环境均可能存在风险，无法保证绝对安全。

## 五、信息保存期限

1. 存储在本地的账号绑定信息、课程表、考试安排、设置、小组件数据等，一般会保存至你在本应用中删除、解绑、清除缓存或卸载本应用为止。
2. 系统安全存储中的账号密码会保存至你在本应用中清除对应账号、解绑账号，或通过系统能力删除相关数据为止。
3. 本地日志会按文件大小和数量滚动保存，超过限制后自动覆盖；你也可以在设置中清除缓存。
4. 错误诊断服务、更新服务、下载服务、CDN、学校系统或第三方系统产生的服务器日志，由相应服务提供方根据其规则保存。我们会在实现处理目的所必要的最短期限内保存由我们控制的信息。
5. 法律法规、监管要求、争议处理、安全审计或故障排查要求另有保存期限的，从其要求。

## 六、我们如何共享、转让和公开披露信息

除以下情形外，我们不会主动向第三方共享你的个人信息：

1. 为实现你主动使用的功能，向学校统一身份认证、教务系统、研究生系统、信息门户或校园网网关提交账号、密码、Cookie、课程请求或认证请求；
2. 为获取公告、配置、更新、安装包或错误诊断服务，向我们的服务器、CDN、更新服务或错误收集服务发送必要请求；
3. 你主动导出日志、复制或分享二维码内容、分享文件、打开外部链接或跳转第三方页面；
4. 根据法律法规、监管要求、司法机关或行政机关依法提出的要求；
5. 为保护用户、我们、学校系统或公众的合法权益，在必要范围内进行处理。

我们不会出售你的个人信息。若未来发生合并、分立、转让、清算或服务运营主体变更，涉及个人信息转移的，我们会要求新的处理方继续受本政策约束，并按法律法规要求履行通知或征得同意。

## 七、第三方服务、SDK 和外部系统

本应用可能访问或集成以下类型的服务、SDK 或系统能力：

1. 学校相关系统：如统一身份认证、教务系统、研究生系统、信息门户、校园网网关等；
2. 我们或项目相关服务：如配置 CDN、公告配置、更新清单、热更新服务、安装包下载服务、项目官网和代码仓库；
3. 错误诊断服务：如我们配置的 GlitchTip/Sentry 兼容错误收集服务；
4. 运行和观测 SDK：如 Expo/React Native 运行时、Expo Updates、Expo Observe、Sentry React Native 等；
5. 系统能力：如相机、系统日历、通知、小组件、WebView、相册、剪贴板、文件和分享面板等。

第三方或外部系统可能有独立的隐私政策和服务规则。你在访问这些系统时，请同时阅读并遵守其规则。

## 八、你的权利和选择

你可以通过以下方式管理你的信息：

1. 在本应用中解绑学校账号、清除校园网账号、清除缓存或删除课程、考试、个性化设置等数据；
2. 在设备系统设置中关闭相机、日历、通知、相册、位置或 Wi-Fi 相关权限；
3. 删除本应用以清除随应用卸载一并删除的本地数据；部分系统安全存储、日历、小组件或系统缓存数据可能需要你在系统设置中进一步清理；
4. 联系我们请求查询、更正、删除或说明个人信息处理情况；
5. 对于你主动导出的日志或分享给第三方的文件，请直接向接收方主张相应权利。

如果你的请求涉及学校系统中的账号、课程、身份或认证数据，请同时联系学校相关系统的管理方处理。

## 九、未成年人保护

本应用主要面向高校用户。若你为未成年人，请在监护人指导下阅读本政策并使用本应用。若监护人发现未成年人未经同意向我们提供个人信息，可以联系我们处理。

## 十、政策更新
